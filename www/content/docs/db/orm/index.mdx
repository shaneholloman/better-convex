Better-Convex ORM brings Drizzle v1’s relational query builder to Convex. You keep the familiar API while gaining Convex’s real‑time, serverless database.

## What You Get

- Drizzle‑style schema and relations (`convexTable`, `defineRelations`, `one`, `many`)
- Type‑safe queries (`findMany`, `findFirst`, `paginate`, `where`, `orderBy`, `with`)
- Mutations (`insert`, `update`, `delete`, `returning`, `onConflictDoUpdate`)
- End‑to‑end TypeScript inference (`InferSelectModel`, `$inferSelect`)
- Convex real‑time subscriptions by default

<Callout icon={<CheckCircle />}>
**API Compatibility**: Better‑Convex mirrors Drizzle v1’s relational query builder surface while adapting to Convex’s document model.
</Callout>

## Where It Differs From SQL ORMs

- `_id` and `_creationTime` are auto‑created
- Fields are nullable by default (`text()` → `string | null`)
- `columns` projection and string operators run post‑fetch
- `with` is relation loading (no SQL joins)
- Cursor pagination needs an index on the primary `orderBy` field

## Quick Example

```ts title="convex/schema.ts"
import { convexTable, defineRelations, text, boolean, id, extractRelationsConfig } from 'better-convex/orm';

export const users = convexTable('users', {
  name: text().notNull(),
  email: text().notNull(),
});

export const posts = convexTable('posts', {
  title: text().notNull(),
  content: text().notNull(),
  published: boolean(),
  userId: id('users'),
});

export const ormSchema = defineRelations({ users, posts }, (r) => ({
  users: { posts: r.many.posts() },
  posts: {
    author: r.one.users({ from: r.posts.userId, to: r.users._id }),
  },
}));

export const ormEdges = extractRelationsConfig(ormSchema);
```

```ts title="convex/queries.ts"
import { query } from './_generated/server';
import { createDatabase } from 'better-convex/orm';
import { ormSchema, ormEdges } from './schema';

export const getRecentPosts = query({
  handler: async (ctx) => {
    const db = createDatabase(ctx.db, ormSchema, ormEdges);
    return db.query.posts.findMany({
      where: { published: true },
      orderBy: { _creationTime: 'desc' },
      limit: 10,
      with: { author: true },
    });
  },
});
```
